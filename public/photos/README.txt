Studio gallery images for BigBets Studio
=======================================

Place your real studio photos here as JPEG or WebP files.

Filenames must match src/config/site.ts → gallery[].src (e.g. studio-1.jpeg … studio-6.jpeg).
Use .jpeg or update site.ts if you prefer .jpg or .webp.

The site uses Next.js Image optimization from this folder (URL path /photos/...).
